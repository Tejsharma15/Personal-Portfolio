    BounceUp game

    A 2D game built using c++ library SDL2. Has a points and leaderboard feature. Built from scratch.

    Taken inspiration from the original game BounceUP, it is a single player game where the objective is to clear the level and collect maximum coins also at the same time avoiding obstacles. The time of the player is also counted which will effect the scoreboard. Ball rotation and a simple physics engine helps with the motion of the ball.

    Future scope: This application currently has only one level. Multiple levels can be added, with the difficulty increasing after each level. The physics engine can be made mode complex to include gravity. This would enhance the player experience.
